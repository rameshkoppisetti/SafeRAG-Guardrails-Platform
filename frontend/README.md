# SafeRAG UI

React + Vite frontend for the SafeRAG Guardrails Platform.

## Run

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

The app calls FastAPI through `/api`, proxied by Vite to `http://localhost:8000`.

## Screens

- Identity & ACL panel
- Session list
- Chat with citations
- Document ingestion form
- Health card
